# 介绍

Ananas Manager 是一个基于NVM工具 VsCode的插件，用于管理Node的版本。

> 🍍：一个看起来像热带水果风味的 Node 可视化版本管理工具

## 安装教程

打开 VsCode 插件市场 搜索 Ananas Manager 安装使用

## 使用教程

成功安装插件后 右方菜单栏会展示一个树状图 点击即可管理Node版本
